# PDNK DataCraft
データを扱えるようになりたいソフトウェア、
データクラフトです。
#### csvから計算結果を作成する場合
例えば、以下のようなCSVがあったとします。(Col1~Col3は除く。)
|Col1|Col2|Col3|
|----|----|----|
|add|1|2|
|sub|3|4|

この場合、

|Mode|In1|In2|Out1|
|----|----|----|----|
|add|1|2|3|
|sub|3|4|-1|

という表を含むマークダウンを出力させる場合のコマンドが存在します。
`./datacraft.o report [入力ファイル名].csv [出力ファイル名].md --csv`
